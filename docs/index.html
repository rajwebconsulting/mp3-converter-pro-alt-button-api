Free YouTube to MP3 Converter Pro Button API

(As featured on GitHub)

  • Fully responsive
  • Customizable with color and size options
  • Grab the code below and replace YOUTUBEID with a valid YouTube video ID. That's it!
<iframe rel="nofollow" style="width:300px; height:60px; border:0px; display:block;" scrolling="no" src="https://yoursite.com/iframe?color=green&vid=YOUTUBEID"></iframe>

Example:

You may change the look by editing color parameter and width & height parameters in style as you like.

Predefined color options:

  • blue
  • green
  • purple
  • red
  • yellow
  • indigo
  • pink
  • gray

Also, you can use ANY hex colors, e.g.:

  • 1383FF
  • 2DB94D
  • DF4454
  • FFCB2F

Use hex color parameter without the '#' sign.

Note: Button is fully responsive, and you may use dynamic width and height. Also, feel free to customize the look further by adding border, shadow, and/or roundness to the iframe style.

This Single Button API is free, and there is no limit with the usage.
